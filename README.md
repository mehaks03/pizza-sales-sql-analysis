# Pizza Sales Data Analysis using SQL

## Project Overview
Analyzed a pizza sales database using SQL to extract business insights related to sales performance, customer ordering patterns, and revenue generation.

## Key Analysis Performed
- Unique pizza categories ordered
- Category-wise sales analysis
- Top-selling pizzas
- Revenue by pizza size
- Average order value
- Peak order hours
- Daily sales trends
- Top revenue-generating pizza types

## SQL Concepts Used
- JOINs
- GROUP BY
- HAVING
- Aggregate Functions
- Date Functions
- Sorting and Filtering

## Tools Used
- SQL Server
- GitHub
